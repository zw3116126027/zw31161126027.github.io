Callable接口(了解即可)扩充

  1. 实现Callable接口，需要返回类型

  2. 重写call方法，需要抛出异常

  3. 创建目标对象

  4. 创建执行服务：ExecutorService ser = Executors.newFixedThreadPool(1);
